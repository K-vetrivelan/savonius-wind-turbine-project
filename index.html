Savonius Wind Turbine Project

Experimental Investigation of 3D Printed Savonius Wind Turbine

Project Overview

This project investigates the performance of a 3D printed Savonius wind turbine with different blade twist configurations (0°, 5°, 10°, and 15°). The blades were designed using CAD software and fabricated using PLA material through FDM 3D printing. The objective is to study how blade twist angle influences turbine performance.
