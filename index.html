Savonius Wind Turbine Project

Experimental Investigation of 3D Printed Savonius Wind Turbine

Project Overview

This project investigates the aerodynamic performance of a Savonius wind turbine fabricated using 3D printing technology. Multiple blade configurations were designed and tested to evaluate the effect of blade twist angles on turbine performance. The blades were modeled using CAD software and manufactured using PLA material.

Wind Tunnel 3D Model

This interactive model shows the custom wind tunnel designed for testing the Savonius wind turbine. The tunnel consists of three main sections: inlet section, converging section, and straight test section where the turbine is mounted for aerodynamic testing.
